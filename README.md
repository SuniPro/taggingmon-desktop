## Environments

- node: `22.14.0`
- pnpm: `9.13.2`

## How To Install

### [Rust 사전 설치](https://v2.tauri.app/start/prerequisites/#rust) 필요

- Linux and macOS: `curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh`
- Windows: `나중에 할 때 추가...`

### Run

1. `pnpm i`
2. `pnpm tauri:dev`

<br />

- 빌드 앱 경로: `src-tauri/target/release/bundle`
